个人博客搭建记录

Posted on 2019-09-08

一、准备github账号

账号略

注意：如果在访问github或者通过homebrew安装软件是经常卡住，那还需要做两件事

1）homebrew切换源镜像仓库地址，一般为中科大或者清华的源

2）修改本机host文件，参考https://www.cnblogs.com/lenmom/p/9400680.html

二、安装nvm，npm,node js

1.nvm安装

安装完成新开一个终端后输入 nvm —version检测是否安装成功

##2.安装nodejs (npm会跟着一起安装)
这里安装v8的最新LTS版本
nvm install v8.16.1
安装完成后输入 node -v 查看node是否安装成功；输入npm -v 查看是否安装成功

#三、安装hexo

##1.hexo以及主题安装

##2.配置
这部分内容可以参考: https://hexo.io/docs/deployment.html
修改_config.yml 配置文件，将hexo与个人的github关联起来，同时需要安装hexo-deployer-git插件：
npm install hexo-deployer-git --save

#四、绑定自己的域名
1.在source文件夹下建立一个CNAME文件，里面写入你的域名，例如www.dazhen.top

2.在github的github page设置中，填入自己的域名
